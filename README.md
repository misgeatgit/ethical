# Ethical
A CLI based Ethiopian calendar.

```sh
>ethical
>እሁድ ታኅሣሥ 30 2015
```

## Installation
### Binary
```
cargo build
```
### Deb package
```
cargo install cargo-deb && cargo cargo-deb
```

### Snap
```
snapcraft
```
